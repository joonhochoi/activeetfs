Active ETF Viewer 사용 가이드

본 프로그램은 화면을 크게 좌측 ETF 목록, 중앙 날짜별 종목 구성 차트, 우측 특정일의 종목 구성 비율 테이블로 나누어 구성되었습니다.

1. 좌측 ETF 목록

  • 현재 총 5개 운용사의 액티브 ETF들을 정리해두었습니다. (ETF 목록 순서는 각 운용사 홈페이지 목록 기준)
  • 액티브 ETF 전문 운용사인 TIME과 KoAct는 제작 시점의 전체 ETF 목록을 포함하고 있습니다.
  • KODEX, RISE, PLUS의 경우, 주요 ETF를 포함하여 선별적으로 구성하였습니다.
  • 각 ETF 이름 옆의 별표(★)를 클릭하면 즐겨찾기로 등록되어 목록 상단에서 쉽게 확인할 수 있습니다.

2. 중앙 차트 영역

  • 좌측 목록에서 선택한 ETF의 종목 구성 변화를 차트로 시각화하여 보여줍니다. (데이터가 없는 경우 표시되지 않습니다.)
  • 상단의 ETF 이름을 더블클릭하면 해당 ETF의 운용사 공식 페이지가 웹 브라우저로 열립니다.
  • 기본 기간: 저장된 데이터의 최신일 기준으로 1주일간의 데이터를 보여줍니다.
  • 기간 설정: 상단의 'View' 옆 메뉴를 통해 표시할 종목 수와 조회 기간을 설정할 수 있습니다.
  • Analyze (분석): 해당 기간 중 편입/편출된 종목 내역을 하단 로그창에 출력합니다.
  • Update & 달력:
    • 달력에 파란색 배경으로 표시된 날짜는 데이터가 존재하는 날짜입니다.
    • 데이터를 갱신하거나 새로 받고 싶은 날짜들을 선택한 후 Update 버튼을 누르면 종목 데이터를 수집합니다.
    • 진행 상황은 하단 로그창에 표시됩니다. (실패 시 운용사 데이터 부재 또는 구조 변경 가능성 있음)
  • 날짜 선택: 차트 내부를 클릭하면 클릭 지점과 가장 가까운 날짜의 데이터가 우측 테이블에 표시됩니다.
  • 편입/편출 종목 확인:
    • 로그창에 출력된 편입/편출 종목을 클릭하면, 차트에서 해당 종목의 라인을 강조해줍니다.
    • Top 10 이하 종목은 차트에 잘 보이지 않을 수 있으므로, 종목 갯수를 All로 설정하면 확인 가능합니다.

3. 우측 종목 구성 테이블

  • 기본적으로 저장된 데이터 중 최신일 기준의 종목 구성 비율을 보여줍니다.
  • 중앙 차트에서 특정 날짜를 클릭하면 해당 일자의 구성으로 변경됩니다.
  • 각 종목의 이름을 클릭하면 차트에서 해당 종목의 라인을 강조해줍니다.
  • 테이블 하단에 위치한 별도 달력을 통해, 차트와 독립적으로 다른 날짜의 데이터를 확인하고 비교할 수 있습니다.

4. 메뉴 (Menu)

  • Version Check: 추후 앱 자동 업데이트 기능을 위해 마련된 버튼입니다. (현재 비활성화)
  • Update All (1day): 특정일을 선택하여 목록에 있는 모든 ETF의 데이터를 일괄 업데이트합니다.
    • 오전 9시 이후 실행을 권장합니다. (너무 이른 시간 실행 시 이전 일자 데이터가 수집될 수 있음)
  • Exit: 프로그램을 종료합니다.

※ 사용상 유의사항

  • 이 프로그램은 실행 파일과 같은 폴더에 있는 activeetf.db 파일에 모든 데이터를 저장합니다.
  • 프로그램을 사용하는 동안 해당 DB 파일을 삭제하지 않도록 주의해 주세요.
