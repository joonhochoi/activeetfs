# Active ETF Viewer 개발 워크스루

## 개요
이 문서는 Active ETF Viewer 애플리케이션의 핵심 개발 과정과 도입된 주요 기능들을 기록한 문서입니다. 초기 프로토타입 구상부터 최근의 아키텍처 개편 및 UI 고도화 작업까지의 흐름을 날짜별 주요 릴리즈 및 변경 사항으로 정리합니다.

## 기술 스택
- **Frontend**: React, TypeScript, Vite, Mantine UI, Tailwind CSS, Apache ECharts
- **Backend**: Rust (**Tauri v2**), SQLite
- **CI/CD**: GitHub Actions (자동 서명 및 릴리즈 파이프라인)

## 📅 날짜별 주요 변경 히스토리

### 🚀 [2026-03-10] Tauri v2 자동 업데이트 시스템 및 빌드 아키텍처 완성
- **Tauri v2 Auto-Updater 인프라 구축**:
  - GitHub Actions를 통한 **자동 업데이트 파이프라인(`latest.json`)**을 완성했습니다.
  - **MSVC 링커 최적화 문제 해결**: 윈도우 빌드 시 업데이트 서명 인식을 방해하던 `__TAURI_BUNDLE_TYPE` 누락 문제를 해결하기 위해 Rust 진입점 아키텍처(`main.rs`로 매크로 이동)를 개편했습니다.
  - **CI/CD 최적화**: GitHub Release 업로드 시 중복 패턴으로 인한 충돌을 해결하고, 자산 업로드 과정을 단일 단계로 통합하여 배포 안정성을 확보했습니다.
- **설치 프로그램(NSIS) 안정화**:
  - 설치 직후 앱 실행 시 권한 문제(Drop-rights)로 인해 WebView2가 충돌하는 현상을 방지하기 위해, NSIS 템플릿을 커스텀하여 "설치 후 바로 실행" 옵션을 제거했습니다.
- **데이터 스크래퍼 및 인코딩 고도화**:
  - **한글 인코딩(Mojibake) 해결**: KoAct, KODEX 등 일부 운용사 데이터 수집 시 한글이 깨지는 현상을 Webview UTF-8 강제 디코딩으로 완벽하게 해결했습니다.
  - **스크래퍼 안정성**: Cloudflare 보안 챌린지 감지 로직을 강화하고 수집 실패 시 폴백(Fallback) 처리를 도입했습니다.
- **플랫폼별 최적화 및 UI 개선**:
  - **macOS 로그 루프 수정**: 로그 파일 생성으로 인한 Tauri dev 모드 무한 재실행 문제를 로그 경로 변경(OS 표준 경로)으로 해결했습니다.
  - **차트 뷰 확대(Zoom) 기능**: ECharts 대시보드에 마우스 휠 및 드래그를 통한 시계열 확대/축소 기능을 추가했습니다.
  - **신규 ETF 지원**: TIME 코스닥 액티브, KoAct 코스닥 액티브 종목을 리스트에 추가했습니다.

### 🚀 [2026-03-09] 시스템 순수 Rust 통합 및 Cloudflare 우회
- **사이드카(Sidecar) 완전 분리 및 저장소 정리**:
  - 기존 Go 언어로 작성되었던 사이드카 디렉토리 트리 전체를 리포지토리에서 제거(`sidecars/` 폴더 삭제)하고 기능들을 온전히 Rust 내부 로직으로 일원화했습니다.
- **KoAct 안티봇(Cloudflare) 우회 기능(WebView Scraper)**:
  - KoAct 데이터 스크래핑 시 발생하는 Cloudflare 403 봇 차단 문제를 해결하기 위해, 백엔드 로직단에서 백그라운드 WebView를 생성하여 페이지 렌더링 후 데이터를 추출하는 우회 기법을 도입했습니다.
- **배포 및 빌드 최적화**:
  - 릴리스 배포 스크립트(`build_release.ps1`)에서 초기 데이터베이스인 `activeetf.db` 패키징 절차를 제거하여, 독립 실행 파일 하나만 배포되도록 빌드 과정을 간소화했습니다. 앱 실행시 필요하다면 내장된 초기 데이터를 자체 생성하게 됩니다.
  - 최신 아키텍처 및 스택 변경 사항을 반영하여 README 문서를 새롭게 재작성했습니다.

### 🚀 [2026-01-31] 아키텍처 대개편 및 일괄 업데이트(Update All) 지원

  - KODEX API 응답의 Null 필드 예외 처리 등 핵심 데이터 정제 로직이 백엔드 내부로 내재화되었습니다.

  - 하루 단위로 모든 운용사/ETF 데이터를 한 번에 가져올 수 있는 전역 일괄 업데이트 기능을 구현했습니다.
  - 독립된 'Update All' 캘린더 다이얼로그 모달을 도입하고, 창 크기 및 레이아웃을 사용성에 맞게 최종 조정했습니다.
- **사이드바 메뉴 확장 및 최적화**:
  - 사이드바 상단에 메뉴 컨트롤 아키텍처를 도입하고 `Help` 기능 등의 확장을 적용했습니다.
- **배포 및 시스템 고도화**:
  - `scripts/build_release.ps1` 스크립트를 최적화하고 적용했으며, 불필요한 레거시 바이너리와 임시 로그 파일 등 미사용 코드를 Git Tracking에서 해제하고 제거했습니다.

## 📊 주요 기능 요약 (Features Synopsis)

### 1. 시계열 자산 비중 대시보드 및 차트 확대
- Echarts 기반의 인터랙티브 차트를 통해 ETF 보유 자산 비중 변화를 추적합니다.
- **차트 확대/축소(Zoom)** 기능으로 세밀한 비중 변화를 시점별로 확인할 수 있습니다.
- **관심 종목(☆) Pin** 기능을 통해 자주 보는 ETF를 빠르게 필터링합니다.

### 2. 포트폴리오 비교 및 분석 (Comparison)
- 당일 기준 ETF 편입 자산 리스트 전부를 출력합니다.
- **과거 데이터 대조**: 우측 날짜 선택기를 조작하여 *과거(기준일) vs 현재* 시점간 포트폴리오 차이를 대조해, 신규 편입 항목(NEW) 및 % 비중 증감을 명확하게 보여줍니다.

### 3. Native Data Scraper & Auto-Updater
- **안티봇 우회 WebView 전용 스크래퍼**: Cloudflare 등의 보안 챌린지를 네이티브 환경에서 우회하여 데이터를 수집합니다.
- **배포판 자동 업데이트**: 앱 실행 시 새 버전 유무를 체크하고 클릭 한 번으로 무중단 업데이트를 수행합니다.
